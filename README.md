# Recommendation-Based-Scholarship-Website
“Recommendation Based Scholarship Website for Bangladeshi Student” is an educational project. Our main motivation is to make a recommendation-based scholarship website so that Bangladeshi students can easily get scholarships in abroad. The system will automate scholarships for students according to their inputs so that both technical and non-technical students can complete the admission process themselves without taking any help form agencies. It is developed of scholarship departments for the group of colleges. Scholarship Officers can add data regarding the scholarships and students. Institution sort out eligible students for the scholarships. It will be an inducement to improve the quality of the study and thereby attaining high standards.
